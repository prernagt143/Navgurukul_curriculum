Lcm/Hcf:

Least common multiple :


Definition: Least Common Multiple(LCM) is a method to find the smallest common multiple between any two or more numbers.
A common multiple is a number which is a multiple of two or more numbers.


Highest common factor :

Definition : Highest Common Factor is the greatest number which divides each of the two or more numbers. HCF is also called the Greatest Common Measure (GCM) and Greatest Common Divisor(GCD).
